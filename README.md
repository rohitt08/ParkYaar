# ParkYaar 🚗

ParkYaar (formerly ParkMate) is a modern, cross-platform parking management application built to simplify the way users find, book, and manage parking spots. 

Built with **Angular 20** and containerized for mobile using **Capacitor**, ParkYaar delivers a seamless experience across web browsers, **iOS**, and **Android** devices.

---

## 🌟 Features

- **Real-time Dashboard:** View total spots, available spots, and occupied spots at a glance.
- **Interactive Parking Layout:** Visually locate available parking spots across different sites, buildings, and floors.
- **Instant Booking:** Click on any available spot to instantly book it by providing vehicle details.
- **Filtering System:** Easily filter parking spots by Site, Building, Floor, and Status.
- **Cross-Platform:** Runs natively on iOS and Android (via Capacitor) and on any modern web browser.

## 🛠️ Tech Stack

- **Frontend Framework:** Angular 20 (Standalone Components)
- **Mobile Container:** Capacitor (for iOS and Android native deployment)
- **Styling:** CSS3 (Custom styling with flexbox/grid layouts)
- **Language:** TypeScript
- **Icons:** Material Icons & FontAwesome

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (v18+) and npm installed. To run the mobile versions, you will also need [Android Studio](https://developer.android.com/studio) (for Android) and/or [Xcode](https://developer.apple.com/xcode/) on a Mac (for iOS).

### 1. Web Development
To run the project locally in your web browser:

```bash
# Install dependencies
npm install

# Start the development server
npm start
```
Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

### 2. Mobile Development (iOS & Android)
To build the project for mobile:

```bash
# Build the Angular web assets
npm run build

# Sync the web code into the native iOS and Android wrappers
npx cap sync

# Open the project in Android Studio (Android) or Xcode (iOS) to run on a device
npx cap open android
npx cap open ios
```

## 🏗️ Project Structure

- `src/app/` - Contains all Angular standalone components, pages (Dashboard, Login), and shared UI elements.
- `src/app/services/` - Contains logic for mocked data management (Parking, User, Transactions).
- `android/` - The native Android project wrapper generated by Capacitor.
- `ios/` - The native iOS Xcode project generated by Capacitor.
